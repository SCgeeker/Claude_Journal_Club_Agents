# OntoLex

> **核心**: the interface between lexica as knowledge representation systems for languages and ontology

## 說明
OntoLex指的是語言的詞彙作為知識表示系統與本體論之間的介面。這個介面涉及語言如何將知識系統化，以及如何在不同的語言之間表示相同的知識。研究OntoLex對於理解語言和本體論之間的關係至關重要。
