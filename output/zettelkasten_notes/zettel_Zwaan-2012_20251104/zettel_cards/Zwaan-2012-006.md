# 物件顏色匹配效應（矛盾的結果）

> **核心**: "Connell obtained faster picture verification responses when the color mismatched than when it matched."

## 說明
Connell的研究發現，與物件方向和形狀的匹配效應相反，參與者對於與句子暗示的顏色不匹配的圖片，反應時間更快。這個結果與心智模擬理論的預測不符，需要進一步的解釋。
