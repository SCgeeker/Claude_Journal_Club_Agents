# 語言情境對對象狀態驗證的影響 (Influence of Linguistic Context on Object State Verification)

> **核心**: "the linguistic context has an impact on the activation of object- state representations."

## 說明
論文發現，語言情境會影響人們對對象狀態的驗證速度。當語言描述的事件與圖片中的對象狀態相符時，驗證速度會更快。這表明語言不僅僅是傳遞信息，還會激活人們對情境的預期，並影響對實際輸入的處理。
