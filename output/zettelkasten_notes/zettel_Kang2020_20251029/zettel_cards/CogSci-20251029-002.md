# 物件狀態 (Object State)

> **核心**: "For example, an object may go through changes due to an external action (e.g., The chef chopped the onion)."

## 說明
物件狀態是指物件在不同時間點所呈現的樣貌或性質。物件可能因為外部作用而發生狀態改變，例如一個洋蔥在被切之前和之後的狀態是不同的。理解物件狀態對於理解事件和情境至關重要。
