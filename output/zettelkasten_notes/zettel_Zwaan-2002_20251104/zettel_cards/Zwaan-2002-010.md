# Neutral Condition in Experiment 2（實驗2中的中性條件）

> **核心**: "The sentences in this condition did not imply anything about the shape of the object (e.g., The ranger heard the eagle in the forest)."

## 說明
實驗2中加入了中性條件，句子不暗示任何關於物體的形狀。這個條件用於比較匹配和不匹配條件，以確定匹配效應是由於匹配條件的促進作用，還是不匹配條件的抑制作用。結果顯示中性條件的反應時介於匹配和不匹配條件之間，無法明確支持促進或抑制的假設。
