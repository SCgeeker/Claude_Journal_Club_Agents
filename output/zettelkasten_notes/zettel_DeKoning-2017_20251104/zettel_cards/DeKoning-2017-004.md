# 內在視覺屬性 (Intrinsic Visual Properties)

> **核心**: "Shape, size, and colour are intrinsic visual properties that distinguish an object from another."

## 說明
內在視覺屬性是指物體本身固有的視覺特性，例如形狀、大小和顏色。這些屬性使物體能夠與其他物體區分開來。
