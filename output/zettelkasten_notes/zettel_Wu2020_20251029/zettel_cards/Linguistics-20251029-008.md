# 量詞的數值 (Numerical Value of Measure Words)

> **核心**: "an M’s value can be anything but 1, thus ¬1."

## 說明
量詞 (M) 的數值可以是除了 1 之外的任何值，表示為 ¬1。這意味著量詞可以直接表示數量，例如 2, 6, 12，或者非數值的計量單位，例如 kilo, meter, liter, bottle 等。
