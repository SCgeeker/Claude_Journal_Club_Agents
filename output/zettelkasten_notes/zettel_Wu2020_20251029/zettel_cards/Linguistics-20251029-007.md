# 分類詞的數值 (Numerical Value of Classifiers)

> **核心**: "a C’s value is numerical and necessarily 1"

## 說明
分類詞 (C) 的數值是數字，且必定為 1。這表示分類詞本身並不直接表示數量，而是作為一種計數單位。
