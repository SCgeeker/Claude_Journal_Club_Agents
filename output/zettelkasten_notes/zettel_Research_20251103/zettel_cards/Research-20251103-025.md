# 語義層面「冗餘」與「必要」的對比 (Redundancy vs Necessity)

> **核心**: "a C’s value is numerical and necessarily 1, while an M’s value can be anything but 1, thus ¬1."

## 說明
此核心說明 C 的數值為 1，導致其在語義上冗餘；而 M 的值各異，說明其在句子中承載實際量化資訊，為必要。此對比解釋了為何在同一結構中 C 可能被視作「無實際價值」的功能。
