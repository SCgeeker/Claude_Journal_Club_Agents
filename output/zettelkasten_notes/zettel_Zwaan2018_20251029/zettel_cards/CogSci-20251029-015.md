# Spacing Effect (間隔效應)

> **核心**: "Learning task in which learning (of words) is spaced over time."

## 說明
間隔效應是指將學習內容分散在一段時間內進行，比集中時間學習效果更好。在這個學習任務中，學習（單詞）是間隔一段時間進行的。
