# 高層次語義處理 (High-level Semantic Processing)

> **核心**: "When an interference technique was used that targeted high-level semantic processing (Experiment 3) however the match effect vanished."

## 說明
高層次語義處理涉及對詞語、句子和篇章的意義進行整合和理解。它不僅包括對單個詞語的意義進行提取，還包括對詞語之間的關係進行分析和推理。本研究中，透過干擾高層次語義處理來檢驗形狀匹配效應是否依賴於這種處理過程。
