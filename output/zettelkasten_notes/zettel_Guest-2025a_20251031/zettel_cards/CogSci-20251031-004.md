# 編程技能退化問題

> **核心**: "Outsourcing Programming to Companies: I can use AI for programming experimental paradigms and statistical analyses"

## 說明
作者認為將編程任務外包給AI是一種「反開源科學的倒退」。心理學家不再需要學習工程技能，而AI生成的代碼缺乏正式規格，屬於黑箱系統。更嚴重的是，輸出代碼將是抄襲的，比自己編寫更耗時檢查合規性，並且違反了開源性原則。這代表了一種專業能力的沙漠化現象，心理學家失去了對研究工具的控制權，也失去了驗證和理解分析過程的能力，最終依賴於無法審查的企業黑箱系統。
