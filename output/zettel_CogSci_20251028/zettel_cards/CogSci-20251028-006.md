# Prompting (提示)

> **核心**: Prompting 是一种通过书面指令引导 LLM 采用特定身份、立场或社会角色的技术。

## 說明
Prompting 的目的是使 LLM 能够模拟特定人群或个人的行为，从而扩展认知科学研究的范围。通过精心设计的提示，研究人员可以操纵 LLM 的输出，以探索不同的情境和假设。
