# 先驗與後驗信念分佈

> **核心**: "In inference we update prior beliefs p(Z) to posterior beliefs p(Z|u) given observed input u."

## 說明
此公式是 Bayesian 推斷的核心：先驗分佈（對各潛在意圖的初始信心）在觀測字串後調整為後驗分佈。  
閱讀成本的關鍵在於這一更新的幅度，亦即信念分佈的「位移」量。
