# Zettelkasten 卡片生成 - 自訂需求

此檔案定義生成 Zettel 卡片時的額外需求。
系統會自動載入此檔案（除非使用 --no-custom 跳過）。

---

## 卡片撰寫風格

- 每張卡片以「一句話核心概念」清晰開頭
- 使用具體範例說明抽象概念
- 在「說明」區塊標註概念的認知層次（感知→理解→應用→分析→評估→創造）

## 連結建立原則

- 優先建立與現有知識庫概念的跨論文連結
- 標註連結類型的認知關係（基礎知識、延伸概念、對比觀點）
- 同一論文內的卡片應形成連貫的知識網絡

## 術語處理

- 專業術語首次出現時附上英文原文
- 重要概念標註領域歸屬（如：認知科學、心理學、神經科學）

## AI 筆記要求

- 提出至少一個批判性問題
- 指出概念的應用限制或潛在爭議
- 連結到相關卡片以建立思考脈絡


## 專有名詞翻譯對照（必須嚴格遵守）

**重要指示**：以下術語必須使用指定的中文翻譯，格式為「中文（English）」，例如「體現認知（Embodied Cognition）」。

| 英文術語 | 中文翻譯 |
|----------|----------|
| Embodied Cognition | 體現認知 |
| Perceptual Symbol | 知覺符號 |
| Amodal Symbol | 非模態符號 |
| Mental Simulation | 心智模擬 |
| Grounded Cognition | 紮根認知 |
| Sensorimotor | 感覺運動 |
| Affordance | 可供性 |
| Modality-specific | 模態特定 |

**注意**：若遇到表中未列出的專業術語，請保持「中文翻譯（English）」的格式。
