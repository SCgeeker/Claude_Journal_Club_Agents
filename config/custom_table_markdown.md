# Markdown 表格處理指引

本檔案指導 LLM 如何處理已轉換為 Markdown 格式的學術論文表格。

---

## Markdown 表格格式

表格會以標準 Markdown 語法呈現：

```markdown
**Table 1: 表格標題**

| 欄位1 | 欄位2 | 欄位3 |
|-------|-------|-------|
| 值1   | 值2   | 值3   |
| 值4   | 值5   | 值6   |

*註：表格註腳*
```

---

## 處理優勢

Markdown 格式的優點：
- **人類可讀**：格式直觀，便於快速瀏覽
- **版本控制友善**：純文字格式，適合 Git 追蹤
- **渲染兼容**：大多數平台可直接渲染

---

## 常見表格類型

### 1. 描述性統計表

```markdown
| Variable | N | M | SD | Range |
|----------|---|---|-------|-------|
| Age | 120 | 25.3 | 4.2 | 18-35 |
| RT (ms) | 120 | 456 | 89 | 312-678 |
```

分析重點：樣本特徵、變異程度

### 2. 推論統計表

```markdown
| Predictor | B | SE | β | t | p |
|-----------|---|----|---|---|---|
| (Intercept) | 2.34 | 0.45 | - | 5.20 | <.001 |
| Condition | 0.89 | 0.23 | .42 | 3.87 | .002 |
```

分析重點：顯著預測變項、效果方向

### 3. 實驗條件比較表

```markdown
| Condition | Accuracy (%) | RT (ms) | d' |
|-----------|--------------|---------|-----|
| Easy | 94.2 (3.1) | 423 (67) | 2.8 |
| Hard | 78.5 (8.4) | 589 (102) | 1.9 |
```

分析重點：條件間差異模式

---

## 分析指引

### 數據解讀

1. **識別主要比較**
   - 哪些條件/組別被比較？
   - 依變項是什麼？

2. **評估效果大小**
   - 數值差異的絕對量
   - 相對變化百分比

3. **判斷統計顯著性**
   - p < .05 標記
   - 信賴區間是否重疊

### 格式處理

當 Markdown 表格有對齊問題時：
- 依據欄位標題推斷數值歸屬
- 若無法確定，標註「格式不明」

## 輸出格式

請以下列格式摘要表格內容：

```markdown
### Table X: [標題]

**表格類型**：[描述性/推論性/比較性]

**關鍵數據**：
| 指標 | 數值 | 意涵 |
|------|------|------|
| [指標1] | [數值] | [說明] |
| [指標2] | [數值] | [說明] |

**主要發現**：
[2-3 句話總結表格核心訊息]

**方法論備註**：
[若有值得注意的統計方法或樣本特性]
```

## 範例

### 輸入 Markdown

```markdown
**Table 3: Correlation matrix for key variables**

| Variable | 1 | 2 | 3 | 4 |
|----------|---|---|---|---|
| 1. Working memory | - | | | |
| 2. Processing speed | .45** | - | | |
| 3. Attention | .52** | .38* | - | |
| 4. Task performance | .61** | .42** | .55** | - |

*Note: N = 85. * p < .05, ** p < .01*
```

### 輸出摘要

```markdown
### Table 3: Correlation matrix for key variables

**表格類型**：相關矩陣（描述性統計）

**關鍵數據**：
| 指標 | 數值 | 意涵 |
|------|------|------|
| WM-Task | r = .61** | 最強相關 |
| Attention-Task | r = .55** | 次強相關 |
| Speed-Attention | r = .38* | 最弱相關 |

**主要發現**：
工作記憶與任務表現的相關最高（r = .61），顯示工作記憶可能是任務表現的重要預測因子。所有認知變項間均呈正相關（r = .38 至 .61），符合認知能力正向流形（positive manifold）的預期。

**方法論備註**：
樣本量 N = 85，使用雙尾 Pearson 相關。相關係數達中至高強度。
```

## 處理複雜表格

### 合併儲存格

若原始表格有合併儲存格（跨欄/跨列），Markdown 可能無法完美呈現。請：
1. 描述整體結構
2. 列出可辨識的數據
3. 標註「原表格含合併儲存格」

### 多層標題

```markdown
|  | Exp 1 |  | Exp 2 |  |
|--|-------|--|-------|--|
| Cond | M | SD | M | SD |
```

處理方式：
- 明確說明層級結構
- 分開報告各實驗結果

---

## 注意事項

1. **保持客觀**：僅報告表格呈現的數據
2. **標示不確定性**：若格式模糊，明確說明
3. **交叉驗證**：若表格數據與內文不符，標註差異
4. **單位一致**：確認並報告測量單位
