## ✅ 導入前檢查清單

使用本清單確保導入順利進行：

```
準備階段:
  [ ] 閱讀 BATCH_B1_EXECUTIVE_SUMMARY.md
  [ ] 確認導入決策 (建議: 導入所有 40 篇)
  [ ] 準備環境 (CrossRef API、計算資源)

  [ ] 驗證評分 9-10 的論文 (2篇)
  [ ] 確認前 3 篇論文的 PDF 可用性
  [ ] 測試導入流程

導入階段:
  [ ] 階段 1: 導入評分 6-10 的論文 (2小時)
  [ ] 階段 2: 批量導入評分 4-5 的論文 (3小時)
  [ ] 階段 3: 驗證和報告 (1小時)

質量控制:
  [ ] 執行 python check_quality.py
  [ ] 執行 python kb_manage.py hybrid-search [測試詞]
  [ ] 生成最終統計報告
  [ ] 備份知識庫
```
