## 📞 遇到問題怎麼辦

### 導入失敗
```
查看步驟:
1. 檢查 output/batch_B1_sync_result.json
2. 查看錯誤日誌
3. 檢查 BibTeX 格式（確保 UTF-8）
4. 驗證知識庫連接
5. 查看具體 entry key 的錯誤
```

### 匹配不正確
```
查看步驟:
1. 檢查 threshold 參數（預設 0.8）
2. 檢查標題是否被 {{}} 正確處理
3. 嘗試 --dry-run 查看匹配結果
4. 必要時調整 strategy 參數
5. 參考 BATCH_B1_PAPERS_TABLE.md 檢查已知論文
```

### 性能問題
```
查看步驟:
1. 檢查知識庫大小（sqlite3 index.db '.tables'）
2. 減少 --workers 參數
3. 確認 CPU/記憶體充足
4. 查看是否有其他程序佔用資源
5. 嘗試分批處理小規模數據
```
